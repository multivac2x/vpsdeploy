# Story 6.1: .env.deploy.example Completeness

**Epic:** Epic 6 - Documentation and Configuration Validation
**Priority:** Must-have

## Description

Verify the example configuration file includes all required and optional variables.

## Acceptance Criteria

- [x] File exists at `.env.deploy.example`
- [x] Includes all required variables: `VPS_USER`, `VPS_IP`, `DOMAIN`, `DOMAIN_TYPE`
- [x] Includes conditional required: `DOMAIN_PORT` (with comment: required if dynamic)
- [x] Includes all optional with defaults: `SSH_KEY`, `SSH_PORT`, `VPS_BASE_PATH`, `VPS_APPS_PATH`, `BUILD_CMD`, `BUILD_OUTPUT`, `PM2_APP_NAME`
- [x] Each variable has descriptive comment
- [x] Template is copy-paste ready: `cp .env.deploy.example .env.deploy`

## Dependencies

None
