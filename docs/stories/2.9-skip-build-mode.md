# Story 2.9: Skip-Build Mode

**Epic:** Epic 2 - deploy.sh Core Functionality Testing
**Priority:** Should-have

## Description

Verify `--skip-build` uses existing build output without rebuilding.

## Acceptance Criteria

- [ ] With `--skip-build`, build phase is skipped
- [ ] Script verifies `BUILD_OUTPUT` directory exists locally
- [ ] If missing, error: "Build output directory '${BUILD_OUTPUT}/' not found. Run without --skip-build first."
- [ ] Deploy phase proceeds with existing output

## Dependencies

Story 2.3

## Testing Notes

```bash
# Create build output
npm run build

# Test skip-build
./deploy.sh --skip-build --dry-run
echo $?  # Should be 0

# Test without build output
rm -rf out/
./deploy.sh --skip-build
echo $?  # Should be 1
```
