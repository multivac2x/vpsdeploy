# Story 1.3: Configuration Validation - Missing .env.deploy

**Epic:** Epic 1 - setup-vps.sh Core Functionality Testing
**Priority:** Must-have

## Description

Script should detect missing configuration file and provide clear guidance.

## Acceptance Criteria

- [x] When `.env.deploy` does not exist, script exits with code 1
- [x] Error message clearly states that `.env.deploy` is missing
- [x] Error message includes template showing required variables
- [x] No remote commands are executed when config is missing

## Dependencies

Story 1.1

## Testing Notes

```bash
# Ensure .env.deploy does not exist
rm -f .env.deploy

# Run script (dry-run to avoid any remote actions)
./setup-vps.sh --dry-run
echo $?  # Should be 1

# Check error message includes template
```
