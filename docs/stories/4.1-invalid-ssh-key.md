# Story 4.1: Invalid SSH Key

**Epic:** Epic 4 - Error Handling and Edge Cases
**Priority:** Should-have

## Description

Verify script handles non-existent or invalid SSH key.

## Acceptance Criteria

- [x] `SSH_KEY` points to non-existent file in `.env.deploy`
- [x] SSH connection attempt fails with "Permission denied" or "Invalid key"
- [x] Script exits with clear error: "Cannot connect... Check your SSH key"
- [x] No changes made to VPS

## Dependencies

None (can test with dry-run or real VPS)

## Testing Notes

```bash
cat > .env.deploy << 'EOF'
VPS_USER=deploy
VPS_IP=your.vps.ip
DOMAIN=example.com
DOMAIN_TYPE=static
SSH_KEY=/nonexistent/key.pem
EOF

./setup-vps.sh --dry-run
echo $?  # Should be 1

# Error should mention SSH key check
```
