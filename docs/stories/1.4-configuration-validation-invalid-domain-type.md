# Story 1.4: Configuration Validation - Invalid DOMAIN_TYPE

**Epic:** Epic 1 - setup-vps.sh Core Functionality Testing
**Priority:** Must-have

## Description

Validate that `DOMAIN_TYPE` must be either `static` or `dynamic`.

## Acceptance Criteria

- [x] With `DOMAIN_TYPE=invalid`, script exits with code 1
- [x] Error message: "DOMAIN_TYPE must be 'static' or 'dynamic'"
- [x] No remote commands executed

## Dependencies

Story 1.1

## Testing Notes

```bash
cat > .env.deploy << 'EOF'
VPS_USER=deploy
VPS_IP=164.90.xxx.xxx
DOMAIN=example.com
DOMAIN_TYPE=invalid
EOF

./setup-vps.sh --dry-run
echo $?  # Should be 1
```
