# Story 1.28: SSH Connection Failure

**Epic:** Epic 1 - setup-vps.sh Core Functionality Testing
**Priority:** Must-have

## Description

Verify clear error when VPS is unreachable.

## Acceptance Criteria

- [x] With invalid `VPS_IP` or SSH service down
- [x] Script attempts SSH connection with timeout (10s) and batch mode
- [x] On failure, exits with code 1
- [x] Error message: "Cannot connect to ${VPS_USER}@${VPS_IP}:${SSH_PORT}"
- [x] Suggests checking: SSH key, VPS_USER, VPS_IP, SSH_PORT
- [x] No partial changes made (registry not updated, directories not created)

## Dependencies

Story 1.1

## Testing Notes

VPS_IP=255.255.255.255
