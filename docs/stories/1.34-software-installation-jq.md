# Story 1.34: Software Installation - jq

**Epic:** Epic 1 - setup-vps.sh Core Functionality Testing
**Priority:** Must-have

## Description

Verify jq installation for JSON processing.

## Acceptance Criteria

- [x] If `jq --version` fails, script installs via `sudo apt install -y jq`
- [x] After install, `jq --version` succeeds
- [x] jq is available for registry manipulation

## Dependencies

Story 1.10 (actual execution)

## Testing Notes

```bash
# On VPS without jq
ssh deploy@vps "sudo apt remove -y jq"
./setup-vps.sh --dry-run --verbose

# Should show: sudo apt install -y jq

./setup-vps.sh
ssh deploy@vps "jq --version"
# Should succeed

# Second run should skip jq install
./setup-vps.sh --verbose 2>&1 | grep jq
# Should show version or skip message
```
