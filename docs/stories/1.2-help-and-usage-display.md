# Story 1.2: Help and Usage Display

**Epic:** Epic 1 - setup-vps.sh Core Functionality Testing
**Priority:** Must-have

## Description

Ensure the script displays proper help information and handles invalid options gracefully.

## Acceptance Criteria

- [x] `./setup-vps.sh --help` displays usage information with examples
- [x] `./setup-vps.sh --help` exits with code 0
- [x] `./setup-vps.sh --invalid-option` displays error message and exits with code 1
- [x] Help text includes all subcommands (setup, status, remove) and flags (--dry-run, --verbose)

## Dependencies

Story 1.1

## Testing Notes

```bash
# Test help
./setup-vps.sh --help
echo $?  # Should be 0

# Test invalid option
./setup-vps.sh --invalid-option
echo $?  # Should be 1
```
